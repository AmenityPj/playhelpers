How To Use
==========

There are various ways to Get Started:

-  Online Mode

   -  Website |image4| can be used

-  Offline Mode (Requires Download / Cloning of the Repo)

   -  Code (as a library) can be directly imported from
      ``playhelpers/ph_*.py`` using any IDE
